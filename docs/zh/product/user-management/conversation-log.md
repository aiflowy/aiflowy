# 对话记录（商业版）功能详解

本文档详细介绍 AIFlowy 平台中的对话记录和对话分析功能（商业版），帮助您全面了解如何查看、分析和管理用户与智能体的对话数据，以及生成用户画像。

## 一、功能概述

### 1.1 什么是对话记录

对话记录是 AIFlowy 平台（商业版）提供的对话数据管理和分析功能，用于查看用户与智能体的历史对话记录，并通过 AI 分析生成用户画像，帮助管理员深入了解用户需求和行为特征。

### 1.2 核心价值

| 价值 | 说明 |
|-----|------|
| 对话历史查看 | 查看用户与智能体的完整对话记录 |
| 用户画像分析 | 通过 AI 分析生成用户画像（职业、兴趣、性格、沟通风格） |
| 对话主题提取 | 自动提取对话主题和核心诉求 |
| 批量分析 | 支持批量分析当天会话 |
| 业务洞察 | 深入了解用户需求，优化智能体服务 |

### 1.3 应用场景

- 用户行为分析
- 智能体服务质量评估
- 用户需求洞察
- 产品优化决策
- 个性化服务推荐

## 二、功能入口

进入 **用户管理 → 对话记录** 页面，可以查看所有用户与智能体的对话会话列表。

## 三、会话列表

### 3.1 列表信息

会话列表展示所有用户与智能体的对话会话，包含以下信息：

| 字段 | 说明 |
|-----|------|
| 对话标题 | 会话的标题（如果没有标题则显示"新对话"） |
| 聊天助手 | 对话使用的智能体名称 |
| 核心诉求 | AI 分析提取的用户核心诉求 |
| 用户画像 | 包含职业、兴趣爱好、性格特点、沟通风格 |
| 创建时间 | 会话创建的时间 |
| 操作 | 分析详情、对话记录、分析 |

### 3.2 查询功能

支持按以下条件查询会话：

| 查询条件 | 说明 |
|---------|------|
| 对话标题 | 搜索会话标题关键词 |

### 3.3 操作功能

| 操作 | 说明 |
|-----|------|
| 分析详情 | 查看会话的分析详情，包括对话主题、核心诉求、用户画像 |
| 对话记录 | 查看完整的对话消息记录 |
| 分析 | 对单个会话进行 AI 分析，生成用户画像 |

## 四、对话分析

### 4.1 分析配置

对话分析使用系统设置的模型进行用户画像分析：

- **分析模型**: 使用系统配置的大模型
- **模型提供商**: 显示当前使用的模型提供商
- **模型名称**: 显示当前使用的模型名称

> **说明**: 分析模型在系统设置中配置，管理员可以修改使用的模型。

### 4.2 分析内容

AI 分析会生成以下内容：

#### 基本信息

| 字段 | 说明 |
|-----|------|
| 对话标题 ID | 会话的唯一标识 |
| 对话标题 | 会话的标题 |
| 聊天助手 | 使用的智能体名称 |
| 创建时间 | 会话创建时间 |
| 分析时间 | 用户画像分析的时间 |

#### 对话分析

| 字段 | 说明 |
|-----|------|
| 对话主题 | 从对话中提取的主题标签 |
| 核心诉求 | 用户的核心需求或问题 |
| 对话摘要 | 对话内容的摘要总结 |

#### 用户画像

| 字段 | 说明 |
|-----|------|
| 职业 | 用户的职业信息 |
| 兴趣爱好 | 用户的兴趣爱好 |
| 性格特点 | 用户的性格特征 |
| 沟通风格 | 用户的沟通方式和风格 |

### 4.3 分析操作

#### 单个会话分析

1. 在会话列表中找到目标会话
2. 点击 **分析** 按钮
3. 系统开始分析会话画像
4. 分析完成后，点击 **分析详情** 查看结果

#### 批量分析今天会话

1. 点击页面右上角的 **分析今天所有会话** 按钮
2. 系统开始分析今天的所有会话
3. 分析完成后刷新列表查看结果

## 五、对话记录查看

### 5.1 功能说明

对话记录查看功能用于查看用户与智能体的完整对话消息，包括：

- 用户发送的消息
- 智能体回复的消息
- 表单交互消息
- 消息发送时间

### 5.2 查看步骤

1. 在会话列表中找到目标会话
2. 点击 **对话记录** 按钮
3. 进入对话记录详情页面
4. 查看完整的对话消息列表

### 5.3 消息类型

| 类型 | 说明 |
|-----|------|
| 用户消息 | 用户发送给智能体的消息 |
| 助手消息 | 智能体回复的消息 |
| 表单消息 | 表单交互相关的消息 |

### 5.4 对话记录信息

对话记录页面显示以下信息：

- **对话标题**: 会话的标题
- **智能体名称**: 对话使用的智能体
- **创建时间**: 会话创建的时间
- **消息列表**: 完整的对话消息，按时间顺序排列

## 六、分析详情查看

### 6.1 查看步骤

1. 在会话列表中找到目标会话
2. 点击 **分析详情** 按钮
3. 在弹窗中查看分析详情

### 6.2 详情内容

分析详情弹窗包含三个部分：

#### 基本信息卡片

显示会话的基本信息：
- 对话标题 ID
- 对话标题
- 聊天助手
- 创建时间
- 分析时间

#### 对话分析卡片

显示对话分析结果：
- 对话主题（标签形式展示）
- 核心诉求
- 对话摘要

#### 用户画像卡片

显示用户画像信息：
- 职业
- 兴趣爱好
- 性格特点
- 沟通风格

### 6.3 无数据分析

如果会话尚未进行分析或分析失败：
- 显示"暂无画像数据"提示
- 提供 **开始分析** 按钮，点击后开始分析

## 七、最佳实践

### 7.1 分析策略建议

- **定期分析**: 定期分析用户对话，了解用户需求变化
- **批量分析**: 使用批量分析功能提高效率
- **重点关注**: 重点关注高频用户和重要客户的对话
- **趋势分析**: 分析用户画像的变化趋势

### 7.2 用户画像应用

- **个性化服务**: 根据用户画像提供个性化服务
- **智能体优化**: 根据用户需求优化智能体配置
- **产品改进**: 根据用户反馈改进产品功能
- **营销策略**: 根据用户特征制定营销策略

### 7.3 数据管理建议

- **定期清理**: 定期清理过期的对话记录
- **数据备份**: 定期备份重要的对话数据
- **权限控制**: 限制对话记录的访问权限
- **合规管理**: 遵守数据隐私相关法规

### 7.4 分析模型选择

- **模型能力**: 选择理解能力强的模型进行分析
- **成本控制**: 平衡分析质量和成本
- **响应速度**: 考虑模型的分析速度
- **准确度**: 选择用户画像准确度高的模型
