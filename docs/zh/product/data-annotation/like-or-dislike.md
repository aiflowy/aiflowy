# 点赞点踩功能详解

本文档详细介绍 AIFlowy 平台中的点赞点踩功能，帮助您全面了解如何收集用户反馈、进行数据标注和优化智能体回答质量。

## 一、功能概述

### 1.1 什么是点赞点踩

点赞点踩是 AIFlowy 平台提供的用户反馈收集和数据标注功能，用于收集用户对智能体回答的评价，并支持管理员对点踩反馈进行标注和优化，最终提升智能体的回答质量。

### 1.2 核心价值

| 价值 | 说明 |
|-----|------|
| 用户反馈收集 | 收集用户对智能体回答的真实评价 |
| 数据标注 | 对点踩反馈进行人工标注，优化回答质量 |
| 数据集构建 | 将标注后的数据保存到数据集，用于模型训练 |
| 质量监控 | 监控智能体回答质量，发现和改进问题 |

### 1.3 应用场景

- 智能客服质量监控
- AI 回答质量评估
- 数据集构建和扩充
- 模型训练数据收集

## 二、功能入口

进入 **数据标注 → 点赞点踩** 页面，可以查看所有用户的点赞点踩反馈记录。

## 三、反馈列表

### 3.1 列表信息

反馈列表展示所有用户提交的点赞点踩记录，包含以下信息：

| 字段 | 说明 |
|-----|------|
| 反馈时间 | 用户提交反馈的时间 |
| 反馈用户 | 提交反馈的用户账号 |
| 反馈 | 点赞（赞）或点踩（踩） |
| 用户提问 | 用户向智能体提出的问题 |
| AI回复 | 智能体的回答内容 |
| 反馈原因 | 用户点踩时填写的原因 |
| 标注状态 | 待标注 / 已标注 |
| 操作 | 去标注 / 查看标注 / 删除 |

### 3.2 查询功能

支持以下条件查询反馈记录：

| 查询条件 | 说明 |
|---------|------|
| 标注状态 | 待标注 / 已标注 |
| 关键词搜索 | 搜索用户提问内容 |

### 3.3 反馈类型说明

| 类型 | 标识 | 说明 |
|-----|------|------|
| 点赞 | 赞（绿色标签） | 用户对智能体回答表示满意 |
| 点踩 | 踩（红色标签） | 用户对智能体回答不满意 |

### 3.4 标注状态说明

| 状态 | 标识 | 说明 |
|-----|------|------|
| 待标注 | 待标注（橙色标签） | 反馈尚未进行人工标注 |
| 已标注 | 已标注（灰色标签） | 反馈已完成人工标注 |

## 四、反馈标注

### 4.1 功能说明

对于点踩的反馈，管理员可以进行人工标注，分析用户不满意的原因，并提供更优质的回答。

### 4.2 标注步骤

1. 在反馈列表中找到 **待标注** 的记录
2. 点击 **去标注** 按钮
3. 在弹窗中进行标注：
   - 选择数据集
   - 问题（AI 辅助优化）
   - 答案（AI 辅助优化）
4. 选择操作方式：
   - **完成标注**: 仅修改标注状态，不保存到数据集
   - **保存到数据集**: 将标注后的数据保存到指定数据集

### 4.3 标注表单

| 字段 | 必填 | 说明 |
|-----|-----|------|
| 数据集 | 是 | 选择要保存到的数据集 |
| 问题 | 是 | 优化后的用户问题 |
| 答案 | 是 | 优化后的标准答案 |

### 4.4 AI 辅助优化

标注表单提供 AI 辅助优化功能：
- **问题优化**: 使用 AI 帮助优化用户问题的表述
- **答案优化**: 使用 AI 帮助优化回答内容

### 4.5 查看标注

对于 **已标注** 的记录：
1. 点击 **查看标注** 按钮
2. 以只读模式查看标注内容
3. 无法修改已标注的内容

## 五、反馈管理

### 5.1 删除反馈

1. 在反馈列表中找到需要删除的记录
2. 点击操作栏的 **...** 按钮
3. 选择 **删除**
4. 确认删除操作

> **注意**: 删除后不可恢复，请谨慎操作。

### 5.2 分页功能

反馈列表支持分页浏览：
- 每页显示条数：10、20、50、100
- 支持跳转到指定页码
- 显示总记录数

## 六、与数据集的关联

### 6.1 数据集选择

在标注反馈时，可以选择将标注后的数据保存到数据集：

1. 在标注弹窗中选择目标数据集
2. 填写优化后的问题和答案
3. 点击 **保存到数据集**
4. 数据会自动添加到选定的数据集中

### 6.2 数据来源标识

保存到数据集的反馈数据会标记来源类型为 **反馈标注**，便于区分不同来源的数据。

### 6.3 数据用途

标注后的数据可以用于：
- 模型训练和微调
- 智能体知识库扩充
- 回答质量评估
- 用户行为分析

## 七、最佳实践

### 7.1 反馈处理建议

- **及时处理**: 定期查看和处理点踩反馈
- **认真标注**: 仔细分析用户不满意的原因
- **持续优化**: 根据反馈不断优化智能体配置
- **数据积累**: 将优质标注数据保存到数据集

### 7.2 标注质量建议

- **问题优化**: 确保问题表述清晰、准确
- **答案优化**: 提供标准、专业的回答
- **AI 辅助**: 善用 AI 辅助优化功能提升效率
- **一致性**: 保持标注风格和术语的一致性

### 7.3 数据集管理建议

- **分类管理**: 按业务场景创建不同的数据集
- **定期审核**: 定期审核数据集中的数据质量
- **版本管理**: 记录数据集的变更历史
- **备份策略**: 定期备份重要数据集

### 7.4 质量监控建议

- **关注点踩率**: 监控智能体的点踩率变化趋势
- **分析原因**: 分析点踩反馈的常见原因
- **优化提示词**: 根据反馈优化智能体系统提示词
- **调整模型**: 必要时调整模型配置或更换模型
