# bot 接入微信公众号
在 bot 设置页面中，配置微信公众号所需要的信息
![img.png](resource/wechat-1.png)
> 点击配置按钮，在弹窗中输入微信公众平台配置的 AppId 、 AppSecret 、 Token 、EncodingAESKey(如果启用加密模式)
> 
![img.png](resource/wechat-2.png)
![img.png](resource/wechat-3.png)
> 配置完成后，将配置弹窗中的“接口地址”复制，进入微信公众平台 → 设置与开发 → 开发接口管理 → 服务器配置
